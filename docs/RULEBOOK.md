# Cigga Go 规则书

## 概述

Cigga Go 是一款以中国香烟文化为主题的轮抽卡牌游戏。收集烟盒、凑品牌和产地 Combo、提防假烟、巧用道具——3 轮比拼，总分最高者赢。

> 3-7 人 · 10-25 分钟 · 14+
>
> 扩展包支持 8 人

> 8 人需要扩展包。

### 2. 组牌

1. 将所有烟盒卡、5 张假烟卡和 8 张道具牌混在一起
2. 充分洗牌
3. 按上表的"三轮总用牌"数量随机抽出本局用牌，剩余牌放回盒中
4. 再次洗牌，作为本局牌堆

### 3. 分发参考卡

每人拿一张计分参考卡（不够可以共享），准备好记分纸和笔。

## 游戏流程

一局游戏共 **3 轮**，每轮流程相同。三轮总分最高的玩家获胜。

### 传牌方向

| 轮次 | 方向 |
|:----:|:----:|
| 第 1 轮 | 向左 ← |
| 第 2 轮 | 向右 → |
| 第 3 轮 | 向左 ← |

### 每轮步骤

#### 1. 发牌

从牌堆中给每人发对应数量的牌。拿到手后看牌，不要给别人看。

> 蓝色卡背 = 道具牌，拿到手就知道了。

#### 2. 选牌

所有玩家**同时**从手牌中选择 **1 张**，**扣在自己面前**。

#### 3. 传牌

将剩余手牌整叠传给对应方向的下家。

#### 4. 翻牌

所有玩家**同时翻开**刚才选的那张牌，展示给所有人看。

#### 5. 重复

拿到下家传来的新手牌 → 选 1 张 → 传牌 → 翻牌……一直重复，直到手中只剩 **2 张**。

#### 6. 最后抉择

从剩下的 2 张中：
- **留 1 张**（扣在面前，暂不公开）
- **弃 1 张**，正面朝上放在桌面中央，所有人可见

#### 7. 抢烟

1. 从剩余牌堆顶**翻开 1 张牌**
2. 所有人**同时伸手拍牌**——先拍到的人拿走
3. 抢到的牌**计入本轮得分**

> **小心：** 翻出来的如果是假烟，先拍到的人要吃 -5 分。手快不一定是好事！

#### 8. 计分

所有人翻开本轮收集的所有牌，按计分规则结算（详见下方）。将本轮得分记在记分纸上。

#### 9. 清场

所有卡牌回收洗入牌堆，开始下一轮。

## 计分

### 计分顺序

```
① 加总所有烟盒卡的基础分
② 加上所有 Combo 加分
③ 加上道具牌效果
④ 如果触发大满贯 → 以上总和 ×2
⑤ 减去假烟的 -5 分（每张）
⑥ 得出本轮得分
```

### 基础分

| 稀有度 | 基础分 |
|:------:|:-----:|
| ★★ | 2 |
| ★★★ | 3 |
| ★★★★ | 5 |
| ★★★★★ | 7 |

### Combo 一览

| Combo | 条件 | 加分 |
|-------|------|:----:|
| **对子** | 同品牌 ×2 | +5 |
| **三条** | 同品牌 ×3 | +12 |
| **同乡** | 同产地 ×2 | +3 |
| **老乡见老乡** | 同产地 ×3 | +8 |
| **粗细搭配** | 手中同时有常规、细支、中支三种规格 | +4 |
| **杂货铺** | 6 张牌品牌全不同 且 产地全不同 | +8 |
| **大满贯** | 同品牌 ×3，且三张各为不同规格（常规+细支+中支） | 总分 ×2 |

**叠加规则：** 所有加分型 Combo 可以自由叠加，同一组牌可以同时触发多个 Combo。大满贯的 ×2 在所有加法结算完毕后才生效。

### 计分示例

**示例 1：稳扎稳打**

本轮保留 7 张牌：芙蓉王(湖南) ×2、云烟(云南) ×1、玉溪(云南) ×1、利群(浙江) ×1、黄鹤楼(湖北) ×1、南京(江苏) ×1，全部 ★★★

- 基础分：3 × 7 = 21
- 对子（芙蓉王 ×2）：+5
- 同乡（云南 ×2：云烟+玉溪）：+3
- **本轮得分：21 + 5 + 3 = 29 分**

**示例 2：Combo 狂魔**

本轮保留 7 张牌：芙蓉王(湖南/★★) ×3（分别是常规、细支、中支）、白沙(湖南/★★) ×1、黄山(安徽/★★★) ×1、散牌 ★★★ ×2

- 基础分：2×3 + 2 + 3 + 3×2 = 16
- 三条（芙蓉王 ×3）：+12
- 老乡见老乡（湖南 ×4，取3张触发）：+8
- 粗细搭配（三种规格齐了）：+4
- 道具 Zippo（最高 Combo 三条 +5）：+5
- 小计：16 + 12 + 8 + 4 + 5 = 45
- 大满贯（芙蓉王 ×3，三种规格各一）：45 × 2 = **90 分！**

## 假烟卡

牌堆中混有 **5 张假烟卡**。

- **外观：** 卡背与普通烟盒卡完全一样。正面看起来像高分卡（知名品牌、高星级），但标有 **「假」** 字
- **效果：** 每张假烟 **-5 分**，在所有 Combo 和大满贯结算完毕后扣除
- **社交名场面：** 第一眼看到"中华 ★★★★★"会兴奋，仔细一看才发现是假的。你可以自己吃亏，也可以传给下家让别人踩雷——计分时追查"谁传的假烟"是游戏最快乐的时刻之一

## 道具牌

道具牌使用**蓝色卡背**，发牌时就能认出来。选道具 = 少选一张烟盒卡，这是真实的代价。

**每人每轮最多保留 1 张道具牌。**

| 道具 | 效果 |
|------|------|
| **Zippo 打火机** | 你本轮分数最高的单个 Combo +5 |
| **茅台酒** | 指定 1 位玩家，你和 TA 各 +4 |
| **麻将桌** | 你的"老乡见老乡"本轮变为 +13 |
| **澳门赌场** | 抛硬币：正面 +10，反面 -5 |
| **假烟鉴定书** | 免疫 1 张假烟的 -5 分 |
| **黑市通行证** | 从本轮弃牌堆中拿回 1 张牌 |
| **走私渠道** | 与另一位玩家交换 1 张已收集的牌（双方公开） |
| **控烟办罚单** | 指定 1 位玩家，TA 本轮最高 Combo 分数减半（向下取整） |

## 进阶规则：验货

> 建议：第一次玩时跳过此规则。熟悉基础玩法后再加入。

**规则：**
- 每轮每位玩家有 **1 次**验货机会
- 在选牌阶段的任意时刻，指着**任意一位玩家最近选的那张扣着的牌**，喊"验货！"
- 被验货的玩家必须翻开那张牌给所有人看
  - **假烟** → 假烟退回给传牌者，不算验货者的分（验货成功！）
  - **真烟** → 验货失败，验货者本轮 **-3 分**

**技巧：** 观察对手选牌时的表情——选得太快、犹豫太久、忍不住笑……都可能是线索！

## 新手建议

1. **先凑对子再想三条。** 对子（+5）容易凑，三条（+12）需要运气和决心，别太贪。
2. **注意产地颜色。** 卡牌上的产地色块帮你快速辨认同产地的牌——颜色相同就是老乡！
3. **假烟要看仔细。** 拿到好牌别急着高兴，先看看有没有那个「假」字。
4. **杂货铺是保底策略。** 如果想凑的 Combo 被人截了，别慌——凑 6 张全不同品牌且全不同产地也有 +8 分。
5. **道具不是白拿的。** 选道具就少一张烟盒卡，想清楚再拿。
6. **大满贯是传说，不是计划。** 同品牌 ×3 还要三种规格各一，碰上了是惊喜，专门去凑大概率翻车。
7. **注意传牌方向。** 第 2 轮方向反了，你的博弈对手也变了。
8. **抢烟前先想想。** 万一翻出来的是假烟，手快的人要吃 -5 分！
9. **粗细搭配容易凑。** 常规、细支、中支各来一张就有 +4 分，顺手就能拿到。
