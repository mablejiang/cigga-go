## 1. 设计愿景

### 1.1 一句话概括

**Cigga Go 是一款以中国香烟文化为主题的轮抽式聚会卡牌游戏，玩家通过收集烟盒、组合品牌/产地/规格三维 combo 来得分，同时提防假烟、利用道具，在策略和社交之间找到乐趣。**

### 1.2 设计目标

- **社交优先，策略为骨**：游戏的每一个机制都应该自然地引发玩家之间的对话、笑声或对峙。规则只是产生社交的工具，不是目的。
- **教学 2 分钟，深度玩 20 局**：第一局看一遍规则就能玩，但品牌搭配、截牌博弈、假烟心理战提供足够的重玩价值。
- **卡牌本身就是产品**：烟盒图的文化价值、风味文字的情绪价值、特殊工艺的收藏价值——即使不玩游戏，这套卡牌也值得拥有。

### 1.3 核心体验

玩 Cigga Go 的一局应该像一场朋友间的"烟局"：

- 前半段安静选牌，像老烟民挑烟——"这个利群不错"、"云烟留着凑云南套"
- 后半段翻牌算分，像拆盲盒——"我凑了三条芙蓉王！"、"谁给我传的假中华？！"
- 结束时聊的不是规则，而是"你怎么不截我那张玉溪"、"下把我要赌大满贯"

### 1.4 设计灵感

| 灵感来源 | 借鉴了什么 | 没有借鉴什么 |
|----------|-----------|-------------|
| **Balatro** | 品牌/产地/规格多维 combo 的核心概念 | 复杂的乘法得分、单人 roguelike 循环 |
| **Sushi Go Party** | 轮抽机制、同时行动、3-8 人支持 | 过于简单的计分（我们加了大满贯和假烟）|
| **中国烟草文化** | 品牌认知、地域归属、面子经济、社交递烟 | 鼓励吸烟（这是文化游戏，不是烟草广告）|

## 2. 核心参数

| 参数 | 值 | 设计原因 |
|------|-----|---------|
| 玩家人数 | 3-8 人 | 覆盖小聚到大桌。轮抽机制天然支持弹性人数 |
| 游戏时长 | 10-25 分钟 | 快局 10-15 分钟（3-4 人），满桌约 20-25 分钟 |
| 适合年龄 | 14+ | 主题涉及烟草文化，不适合太小的孩子 |
| 教学时间 | 2-3 分钟 | 社交游戏的硬门槛——超过 5 分钟教学就会流失玩家 |
| 重玩性 | 高 | 每局用牌堆子集、轮抽随机性、假烟心理战保证变化 |

## 3. 社交设计总览

这个 section 把社交设计逻辑集中展示。Cigga Go 的每一个机制都经过"这会产生什么样的社交时刻？"的审视——如果一个机制不产生社交，它就应该被简化或移除。后续章节会详细展开每个具体机制的规则细节。

### 3.1 设计哲学：规则服务于社交

Cigga Go 不是一个碰巧有社交元素的策略游戏——它是一个**用策略机制驱动社交的聚会游戏**。

这个区分很重要。它意味着：
- 当"好玩的规则"和"好玩的社交"冲突时，**社交优先**
- 一个产生笑声和争论的机制，比一个"设计精巧但大家闷头算分"的机制更有价值
- 规则的复杂度上限由"是否打断社交节奏"决定，而不是由"是否增加策略深度"决定

### 3.2 社交时刻地图

一局 Cigga Go 的社交分布不是均匀的——它有**安静期**和**爆发期**，像呼吸一样交替，形成节奏。

```
  社交强度
  ▲
  █                                          ████  ← 翻假烟/抢烟/大满贯
  █                                     █████████
  █                                ██████████████
  █            ███████        █████████████████████
  █       █████████████  ██████████████████████████
  █  ████████████████████████████████████████████████
  └──────────────────────────────────────────────────→ 时间
    发牌  选牌  传牌  选牌  传牌 ... 最后抉择  翻牌算分  抢烟
    ←──── 安静策略期 ────→         ←──── 社交爆发期 ──→
```

**安静策略期（选牌+传牌）：**
- 玩家专注于自己的手牌，偶尔偷看别人的表情
- 社交是低声细语式的——"他刚才选得好快，肯定是好牌"
- 这段安静是**必要的**——它为后面的爆发蓄能

**社交爆发期（翻牌+算分+抢烟）：**
- 所有隐藏信息同时揭开，产生密集的反应
- "谁传的假烟"、"大满贯！"、"快抢那张！"
- 每轮结束都以一个高潮收尾

这种节奏避免了两种极端：全程安静（无聊）和全程吵闹（疲惫）。

### 3.3 六大社交引擎

Cigga Go 有六个不同的机制在产生社交，每个针对不同类型的社交体验：

#### 引擎 1：截牌博弈（轮抽固有）

**社交类型：** 心理对抗、默契博弈

轮抽机制天然产生"读人"的社交。你看到一张好牌，但你知道下家也在凑同样的套路：

- "他上轮拿了两张云南，我截不截这张玉溪？"
- "她故意选得很慢，是在犹豫还是在演我？"

这种社交是**安静的、持续的**——不会产生尖叫或大笑，但它让每一次选牌都有了"对人下棋"的意味。

**传牌方向交替**强化了这一点：第1轮你和左邻博弈，第2轮变成右邻，第3轮又回来。每轮的"对手"不同，社交关系在全桌流转。（轮抽机制详见第5节）

#### 引擎 2：假烟心理战

**社交类型：** 欺骗、发现、指责、大笑

假烟是这个游戏**最强的社交引擎**，因为它在每个阶段都产生社交：

| 阶段 | 社交时刻 |
|------|---------|
| 选牌时 | "这张中华……等等，是假的！" → 发现时刻的自嘲 |
| 传牌时 | "我要不要传给下家？" → 欺骗的道德博弈 |
| 翻牌时 | "谁给我的假烟？！" → 全桌追查嫌疑人 |
| 验货时 | "你选牌表情不对，我验货！" → 直接对峙 |

假烟的社交价值在于**信息不对称**——你知道你传了假烟，但下家不知道是你传的。这种不对称自然引发猜疑、辩解、大笑。

（假烟的详细机制见第9节）

#### 引擎 3：抢烟环节

**社交类型：** 物理互动、争夺、即时反应

每轮计分结束后，从剩余牌堆中**翻开 1 张公共牌**，所有人同时拍桌抢。

**规则：**
- 计分完毕后，翻开牌堆顶部 1 张牌
- 所有玩家同时伸手拍牌——**先拍到的人获得这张牌**
- 获得的牌**计入本轮得分**（可能会改变排名）
- 如果翻出假烟——先拍到的人必须吃下 -5 分（惩罚手快的人）

**为什么需要这个环节：**
- 轮抽和算分都是"脑力活动"，抢烟加入**身体互动**，打破节奏
- 全桌同时伸手拍牌是一个天然的"热闹时刻"，自带笑声和混乱
- 假烟可能被翻出增加了"要不要抢"的犹豫——手快可能反而中招
- 从博弈到肢体到大笑，这个 5 秒钟的环节把每轮的结尾推向高潮

**社交场景：**
- 翻出 ★★★★★ 卡 → 全桌争抢 → 手快的人欢呼
- 翻出假烟 → 有人条件反射拍了 → 全桌爆笑 → "你也太快了吧！"
- 两个人同时拍到 → "我先拍的！" "你明明慢了！" → 争论但快乐

#### 引擎 4：道具牌的社交效果

**社交类型：** 合作、攻击、赌博、谈判

道具牌中有一半以上设计了**必须与其他玩家互动**的效果：

| 道具 | 社交动作 | 产生的对话 |
|------|---------|-----------|
| 茅台酒 | 选一个人合作 +4 | "我请你喝酒" → "为什么选他不选我？" |
| 走私渠道 | 和别人强制交换一张牌 | "我换你那张中华" → "我不想换！但规则说必须换……" |
| 控烟办罚单 | 指定一个人削弱 | "这一轮你分最高所以……" → "我们是朋友吗？" |
| 澳门赌场 | 全桌围观抛硬币 | "正面正面正面！" → 全桌倒计时 → 结果揭晓 |

这些道具牌制造了**定向社交**——不是泛泛的"大家都在玩"，而是"你和特定某个人之间发生了一件事"。这种一对一的互动更深刻、更有记忆点。

（道具牌系统见第10节）

#### 引擎 5：最后抉择的信息暴露

**社交类型：** 信息推理、调侃

每轮结束时的"二选一弃一"（公开弃牌）看似是个简单的规则，但它是一个精心设计的社交触发器：

- 你弃掉的牌暴露了你"不想要什么"，间接暴露了你"想要什么"
- "你扔了芙蓉王？你凑到三条了？！" → 计分前的情报分析
- "你扔了 ★★★★★？！" → 可能里面有假烟 → 猜测和讨论

弃牌信息是**公开的但需要解读的**——它不是直接告诉你答案，而是给你一条线索。这种"线索式社交"比直接翻底牌更有趣，因为它留了讨论空间。

#### 引擎 6：大满贯的全桌高潮

**社交类型：** 集体震撼、传说时刻

大满贯（x2）在数值上是一个计分规则，但在体验上它是一个**社交事件**。

因为极难达成，大满贯几乎不会每局出现。但一旦有人在计分时说出"我有大满贯"：
- 全桌会要求验证——"给我看！真的假的？"
- 确认后是集体性的反应——欢呼、叹息、不敢相信
- 这个时刻会成为"传说"——被反复提起，成为下一局的谈资

大满贯的社交价值不在于它出现的那一刻，而在于**它可能出现的每一刻**。每次有人盯着手牌犹豫时，旁边的人都会猜："他是不是在赌大满贯？"

（combo系统见第8节）

### 3.4 社交与规则复杂度的平衡

Cigga Go 的每个环节都经过了"社交-规则"平衡的审视：

| 环节 | 规则复杂度 | 社交属性 | 平衡判断 |
|------|-----------|---------|---------|
| 轮抽选牌 | 低（选1张传） | 中（读人、截牌） | 规则简单，社交自然产生 |
| Combo 计分 | 中（查表加法） | 低（各算各的） | 计分是规则最复杂的部分，但只占每轮 1-2 分钟（详见第8节） |
| 大满贯 | 低（唯一乘法） | **高**（全桌炸锅） | 规则极简（x2），社交回报极高（详见第8节） |
| 假烟揭晓 | 低（-5 分） | **极高**（互相指责+大笑） | 最佳平衡点——规则最简单的机制产生最强社交（详见第9节） |
| 验货（可选） | 低（翻牌 + 判定） | **极高**（bluff + 对峙） | 作为可选规则存在，不增加基础教学负担（详见第9节） |
| 抢烟 | 低（拍牌） | **极高**（肢体互动 + 笑声） | 零规则解释成本，纯物理互动 |
| 最后弃牌 | 低（二选一） | 中（暴露信息） | 简单规则触发信息推理式社交 |
| 道具牌-茅台酒 | 低（选人 +4） | **高**（选谁？为什么？） | 一句话规则驱动深度社交（详见第10节） |
| 道具牌-赌场 | 低（抛硬币） | **高**（全桌围观） | 零策略负担的纯氛围机制（详见第10节） |
| 道具牌-罚单 | 低（指人削弱） | **高**（攻击性社交） | 简单规则但产生强烈情绪反应（详见第10节） |

**关键观察：**
- 社交属性最高的机制（假烟、抢烟、验货）恰恰是规则最简单的
- 规则最复杂的环节（combo 计分）社交属性最低——但它被压缩在每轮 1-2 分钟内，不会长时间打断社交
- 没有任何环节是"规则复杂且社交为零"的——如果出现这种情况，说明设计有问题

### 3.5 社交节奏设计

一局 Cigga Go（3 轮）的社交节奏像三个浪：

**第 1 轮：破冰**
- 玩家还在学习规则，选牌较慢
- 社交来自"发现"——"这个品牌我知道！我爸抽的"
- 假烟第一次出现时的反应最强烈（"还有假烟？！"）（假烟的详细机制见第9节）
- 抢烟环节帮助破冰——所有人伸手的瞬间消除了拘谨

**第 2 轮：竞争**
- 玩家开始有策略，截牌意识增强
- 社交从"发现"变成"博弈"——"别给他那张！他在凑三条！"
- 道具牌的攻击性使用增多（"罚单给你！谁让你上轮最高"）（道具牌系统见第10节）
- 传牌方向改变，新邻居关系产生新对话

**第 3 轮：决战**
- 分数接近，每张牌都关键
- 社交达到顶峰——"求你别截我"、"我赌大满贯了不管了"（combo系统见第8节）
- 计分时最紧张，可能出现逆转
- 最终排名揭晓——赢家庆祝，输家复盘（"就是第二轮那张假烟害我的"）

### 3.6 社交设计的边界

有些"看起来很社交"的机制被有意排除，因为它们会**损害体验**：

| 被排除的机制 | 为什么 |
|-------------|--------|
| **公开讨论/禁止说话轮** | 强制社交适得其反——社交应该自然发生 |
| **投票淘汰某人的牌** | 少数被投票的人体验极差 |
| **交易/自由市场** | 8 人自由交易会让游戏时间失控 |
| **复杂的联盟机制** | 聚会游戏不适合需要"记忆谁和谁结盟"的机制 |
| **过多的攻击性道具** | 1-2 张攻击牌增加刺激，过多会让人不想玩 |

**原则：** 好的社交机制让人们**想继续玩**。如果一个机制让某人觉得"被针对了"或"太复杂了"，它就应该被砍掉，不管它在纸面上多有趣。

## 4. 收藏设计理念

### 4.1 为什么收藏性与游戏性并重

Cigga Go 的卡牌不只是游戏组件——它们是中国烟草文化的微型博物馆。每张卡上的真实烟盒图来自 ciggies.app，本身就有文化和收藏价值。

这意味着 Cigga Go 面向两类用户：
- **玩家**：用这些卡牌来进行轮抽、凑 combo、社交互动
- **收藏者**：被烟盒设计的美学、文化记忆和工艺吸引，即使不玩游戏也值得拥有

这两个目标不矛盾——好的游戏卡牌设计可以同时服务两者。卡面上的品牌名和分数服务于游戏功能，烟盒图和风味文字服务于收藏情绪，而不同稀有度的工艺处理则让两个目标在物理触感上统一。

### 4.2 稀有度的工艺哲学

Cigga Go 的稀有度由价格决定，对应不同的物理工艺处理，让玩家从**指尖**就能感受到差异：

| 稀有度 | 价格区间 | 基础分 | 游戏定位 | 物理感受目标 |
|--------|---------|--------|---------|-------------|
| ★★ | <=15元 | 2 | 基石牌（约 22%），combo 的基础 | 轻薄顺滑，适合快速扇牌 |
| ★★★ | 16-40元 | 3 | 中坚力量（约 54%） | 与 ★★ 类似但略厚实 |
| ★★★★ | 41-80元 | 5 | 进阶牌（约 16%），高分但数量少 | 磨砂触感 + 烫银，拿到手里明显"不一样" |
| ★★★★★ | 81元以上 | 7 | 传说牌（约 8%），极高基础分 | 丝绒覆膜 + 全息 + 限量编号，从指尖就感觉是"好牌" |

**核心原则：** 不同稀有度的卡牌应该在**触感和视觉**上就能区分，拿到传说卡的瞬间应该从指尖就感觉不一样。

**历史/停产卡牌的收藏溢价：** 收藏限定包中的历史卡牌享受 +1 稀有度加成（如原本 ★★ 变为 ★★★），体现停产品牌的收藏溢价。

### 4.3 卡牌作为文化载体

每张 Cigga Go 卡牌承载的不只是游戏数据，更是一段文化记忆：
- 烟盒图来自真实的烟草包装设计，记录了中国几十年的视觉设计演变
- 风味文字（如"国宴标配"、"工人阶级的选择"）为每张卡赋予了情绪和故事
- 规格属性（常规/细支/中支）展现了中国烟草产品形态的多样化趋势

这种文化载体的定位意味着 Cigga Go 的卡牌不会像普通游戏组件那样被"用完即弃"——它们值得被翻看、讨论、收藏。

（详细的 SKU 级工艺规格和扩展包收藏设计见第13节）

## 5. 核心机制：轮抽

### 5.1 为什么选轮抽

轮抽（card drafting）是最适合这个项目的机制，原因：

| 需求 | 轮抽如何满足 |
|------|-------------|
| 3-8 人 | 同时行动，人数弹性天然 |
| 10-25 分钟 | 没有等待，每秒都在做决策 |
| 社交互动 | "读人"——猜下家要什么、决定截不截 |
| 策略深度 | 每次选牌都是多维博弈（自己要什么 vs 给下家留什么） |
| 教学简单 | "选一张，传下去"——一句话规则 |

### 5.2 轮抽如何产生社交

轮抽不是闷头选牌。社交自然发生在这些时刻：

1. **传牌时的心理博弈**："我知道你在凑云南，但这张玉溪我自己也想要……"
2. **同时翻牌时的反应**："你也拿了芙蓉王？那我的对子怎么凑？"
3. **假烟传递**："是谁给我传的这张假中华？" → 全桌笑
4. **截牌讨论**："你为什么不截他那张？你看他现在三条了！"

### 5.3 传牌方向交替

- 第 1 轮：向**左**传
- 第 2 轮：向**右**传
- 第 3 轮：向**左**传

**为什么？** 如果始终向左传，你只和左右两个邻居有博弈关系。方向交替后，每轮的"威胁对象"不同，社交覆盖全桌。这是 Sushi Go 验证过的成熟设计。

## 6. 回合结构

### 6.1 完整一局流程

```
┌─ 准备阶段 ────────────────────────────────────────────┐
│  1. 根据人数从总牌堆中随机抽取本局用牌（见6.2）         │
│  2. 将假烟牌和道具牌混入                               │
│  3. 洗牌                                               │
└───────────────────────────────────────────────────────┘
                          ↓
┌─ 第1轮（向左传）─────────────────────────────────────┐
│  1. 发牌：每人拿到 N 张（见6.2）                      │
│  2. 选牌：所有人同时选 1 张，扣在自己面前              │
│  3. 传牌：将剩余手牌传给左手边的玩家                   │
│  4. 揭牌：所有人同时翻开刚选的那张牌                   │
│  5. 重复 2-4，直到手中只剩 2 张                       │
│  6. 最后抉择：从 2 张中选 1 张保留，1 张弃掉（公开）  │
│  7. 计分：统计本轮所有收集卡牌的总分                   │
│  8. 抢烟：从剩余牌堆翻1张，所有人同时拍桌抢           │
│  9. 清场：所有卡牌回收                                │
└───────────────────────────────────────────────────────┘
                          ↓
              第2轮（向右传）→ 第3轮（向左传）
                          ↓
                  累计三轮总分，最高分胜
```

### 6.2 人数与发牌数

| 人数 | 每人每轮发牌数 | 每轮最终保留 | 每轮总用牌 | 三轮总用牌 |
|------|--------------|-------------|-----------|-----------|
| 3 | 7 | 6 | 21 | 63 |
| 4 | 9 | 8 | 36 | 108 |
| 5 | 8 | 7 | 40 | 120 |
| 6 | 8 | 7 | 48 | 144 |
| 7 | 7 | 6 | 49 | 147 |
| 8 | 7 | 6 | 56 | 168 |

**设计说明：**
- 3 人局发 7 张保留 6 张（而非 9/8），节奏更紧凑，与 7-8 人保持一致
- 4-6 人维持原有配置，经过充分测试验证
- 人数越多每人拿的牌越少，保证每轮时间不会因为人多而变长
- 基础版 147 张牌支持 3-7 人游戏；加入国际扩展包（+21 张 = 168 张）支持 8 人

### 6.3 "最后抉择"设计

每轮结束时手里剩 2 张，选 1 弃 1。弃掉的牌**公开放在桌面**。

**为什么公开弃牌？**
- 信息暴露：所有人都能看到你放弃了什么，推断你保留了什么
- 社交触发："你扔了软中华？？？" "里面有假的你信不信"
- 策略价值：通过观察弃牌推断牌堆里还剩什么

### 6.4 一轮的时间拆解

以 6 人为例（每人 8 张）：

| 步骤 | 时间 | 说明 |
|------|------|------|
| 看牌 + 选牌（第一轮选牌会稍慢） | 约 15 秒/次 x 7 次 | 所有人同时，约 2 分钟 |
| 最后抉择 | 约 10 秒 | 二选一 |
| 计分 | 约 1-2 分钟 | 纯加法，大家各算各的，互相验证 |
| **单轮合计** | **约 4-5 分钟** | |
| **三轮合计** | **约 12-15 分钟** | 加上准备和讨论约 18-22 分钟 |

## 7. 卡牌系统

### 7.1 卡牌类型总览

| 类型 | 数量 | 卡背颜色 | 作用 |
|------|------|---------|------|
| **烟盒卡** | 134 张 | 红色 | 核心卡牌，收集并组合得分 |
| **假烟卡** | 5 张 | 红色（与烟盒相同） | 负分陷阱，社交炸弹 |
| **道具牌** | 8 张 | 蓝色 | 特殊效果，增加策略变化 |
| | **合计 147 张** | | 基础版总牌数 |

### 7.2 烟盒卡属性——三维体系

每张烟盒卡有 **3 个核心维度 + 2 个数值属性**，全部印在卡面上：

| 属性 | 说明 | 对游戏的影响 |
|------|------|-------------|
| **品牌** | 香烟品牌名（中华、玉溪等），共 47 个品牌 | 对子/三条 combo 的匹配依据 |
| **产地** | 生产省份（上海、云南等），共 24 个省份 | 同乡/老乡见老乡 combo 的匹配依据 |
| **规格** | 常规/细支/中支 | 粗细搭配 combo + 大满贯条件 |
| **稀有度** | ★★ 到 ★★★★★（由价格决定） | 决定基础分和卡牌工艺 |
| **基础分** | 2/3/5/7 | 得分基础 |

**三维设计的核心优势：**
- **品牌 + 产地** 是最有文化辨识度的两个维度——"中华是上海的"、"芙蓉王是湖南的"，即使非烟民也有认知
- **规格**（常规/细支/中支）直观可见，对应烟支的物理粗细，容易理解
- 三个维度产生的 combo 组合已经足够丰富，不会让选牌时的认知负担过重

### 7.3 规格（Format）分布

规格是卡牌的第三维度，反映了中国烟草行业近年来的产品形态分化趋势。

| 规格 | 中文 | 牌堆中张数 | 占比 |
|------|------|-----------|------|
| **常规** | 传统标准烟支 | 45 | 33.6% |
| **细支** | 近年流行的纤细款 | 45 | 33.6% |
| **中支** | 介于常规和细支之间 | 44 | 32.8% |

**设计意图：** 三种规格近乎均匀分布（45/45/44），确保"粗细搭配"combo（需集齐三种规格）在合理难度内可达成。

### 7.4 品牌在牌堆中的分布

品牌分布直接影响 combo 的可凑程度，是数值平衡的基础。基础版包含 **47 个国内品牌**。

**核心品牌（每品牌 4-5 张）——容易凑 combo：**

| 品牌 | 产地 | 张数 | 设计意图 |
|------|------|------|----------|
| 中华 | 上海 | 5 | 国民第一品牌，认知度最高 |
| 玉溪 | 云南 | 5 | 云南代表 |
| 芙蓉王 | 湖南 | 5 | 湖南代表 |
| 黄鹤楼 | 湖北 | 5 | 湖北代表 |
| 利群 | 浙江 | 5 | 浙江代表 |
| 云烟 | 云南 | 5 | 云南第二品牌 |
| 七匹狼 | 福建 | 5 | 福建代表 |

这些品牌每个有 4-5 张，意味着**理论上可以凑三条**（同品牌 3 张），但需要在轮抽中与其他玩家竞争。核心品牌通常覆盖了常规/细支/中支三种规格，为大满贯提供可能。

**次核心品牌（每品牌 3-4 张）——能凑对子，三条需要运气：**

娇子（四川）、双喜（广东）、白沙（湖南）、黄山（安徽）、南京（江苏）、黄金叶（河南）、苏烟（江苏）、红塔山（云南）、红双喜（上海）、贵烟（贵州）等。

**稀有品牌（每品牌 2 张）——对子即上限：**

熊猫（上海）、大重九（云南）、和天下（湖南）、黄鹤楼1916（湖北）、南京九五（江苏）、荷花等。

这些品牌只有 2 张，凑对子就是最好情况。但它们往往稀有度高（★★★★-★★★★★），单张基础分就很可观。

**设计原理：** 核心品牌多但基础分低（★★-★★★），稀有品牌少但基础分高（★★★★-★★★★★）。这创造了 combo 派 vs 高分散牌派的策略分歧。

### 7.5 产地分布

产地分布决定了"同乡"combo 的可凑程度。基础版覆盖 **24 个省份**。

| 产地 | 牌堆中张数 | 包含品牌 | 设计意图 |
|------|-----------|---------|----------|
| **云南** | ~18 | 玉溪、云烟、红塔山、大重九 | 最大产地，凑同乡容易，但人人都想凑 |
| **上海** | ~16 | 中华、红双喜、熊猫、牡丹 | 高端烟集中地 |
| **湖南** | ~11 | 芙蓉王、白沙、和天下 | 中等密度 |
| **湖北** | ~9 | 黄鹤楼、黄鹤楼1916 | 品牌集中 |
| **江苏** | ~9 | 南京、苏烟、南京九五 | 品牌集中 |
| **其他省份** | 各 2-7 | 各省代表品牌 | 分散，凑同乡难但凑到有惊喜感 |

**设计意图：** 云南和上海是"安全牌"——牌多容易凑，但也意味着其他人可能也在抢。小省份是"险牌"——牌少凑不到就浪费了，但凑到的话竞争小。

### 7.6 国际品牌的定位

国际品牌（万宝路、骆驼、大卫杜夫、柔和七星、三五、高希霸、登喜路等）**作为独立扩展包**，共 21 张。

**不放入基础版的理由：**
- 基础版主打"中国烟草文化"的主题一致性
- 国际品牌与国内品牌不存在产地 combo 关系，会稀释 combo 体系
- 作为扩展包有独立的收藏价值和新鲜感
- 加入国际扩展包后总牌数达到 168 张，解锁 8 人游戏

## 8. Combo 系统与计分

### 8.1 Combo 总览

| Combo 名 | 英文 | 条件 | 加分 | 出现频率 | 设计意图 |
|----------|------|------|------|---------|----------|
| **对子** | Pair | 同品牌 2 张 | +5 | 高 | 基础 combo，几乎每轮都有人凑到 |
| **三条** | Three-of-a-Kind | 同品牌 3 张 | +12 | 中 | 需要投入，回报跳跃式增长 |
| **同乡** | Hometown | 同产地 2 张 | +3 | 高 | 比品牌容易凑（同产地牌更多） |
| **老乡见老乡** | Old Friends | 同产地 3 张 | +8 | 中 | 稳健策略的核心 |
| **粗细搭配** | Format Clash | 同时拥有常规+细支+中支各至少 1 张 | +4 | 中 | 鼓励收集不同规格，三种均匀分布使其可行 |
| **杂货铺** | Mixed Bag | 6 张全不同品牌且全不同产地 | +8 | 中 | 给"什么 combo 都没凑到"的人一条活路 |
| **大满贯** | Grand Slam | 同品牌 3+ 张且包含常规+细支+中支全部 3 种规格 | 总分 x2 | 极低 | 整局游戏的终极高潮时刻 |

### 8.2 Combo 叠加规则

**所有加法 combo 自由叠加。** 同一组牌可以同时触发多个 combo。

**案例：** 你收集了 3 张芙蓉王（湖南），分别是常规、细支、中支。

- 三条（同品牌 3 张）：+12
- 老乡见老乡（同产地湖南 3 张）：+8
- 粗细搭配（三种规格齐全）：+4
- 3 张基础分假设都是 3（★★★）：9
- **本组总分：9 + 12 + 8 + 4 = 33 分**

由于这 3 张芙蓉王是同品牌且覆盖了全部 3 种规格，还能触发大满贯 → **33 x 2 = 66 分**。

### 8.3 大满贯的详细规则

**触发条件：** 你收集的牌中，存在同一品牌的 3 张以上的牌，且包含常规、细支、中支全部 3 种规格。

**效果：** 本轮你的总分 x2（先算完所有加法 combo，再翻倍）。

**为什么设定为 x2 而不是更高？**
- x2 已经能让分数从"正常"变成"碾压"，足够产生兴奋感
- x3 或更高可能导致"大满贯玩家必赢"，降低其他策略的价值
- 而且大满贯是有风险的——你为了凑它可能牺牲了其他 combo，如果没凑到分数反而低

**为什么极难达成？**

以中华为例，牌堆里的中华卡有 5 张：
- 中华（5支硬盒）— 常规
- 中华（听50支）— 常规
- 中华（细支）— 细支
- 中华（细支6mg）— 细支
- 中华（双中支）— 中支

要凑大满贯需要同品牌 3+ 张且覆盖三种规格。你需要至少拿到 1 张常规 + 1 张细支 + 1 张中支的中华。这要求这些牌都在本轮牌堆里、都经过你手、而且你都选了它们。在 5 人游戏中，模拟结果显示大满贯出现概率约 1.9%。

**模拟验证的出现频率：**

| 人数 | 大满贯概率 | 目标区间 |
|------|-----------|---------|
| 3人 | 0.5% | 0.5-2% |
| 5人 | 1.9% | 0.5-2% |
| 7人 | 0.8% | 0.5-2% |

**社交效果：** 正因为极难，一旦有人达成，全桌会自发欢呼/惊叹。这种"传说时刻"会成为这局游戏被记住的原因——"就是那局有人打出大满贯的"。

### 8.4 计分流程

每轮结束时：

```
1. 每张烟盒的基础分加在一起
2. 检查所有品牌 combo（对子/三条），加上对应加分
3. 检查所有产地 combo（同乡/老乡见老乡），加上对应加分
4. 检查粗细搭配和杂货铺，加上对应加分
5. 加上道具牌的加分（如果有）
6. 检查是否触发大满贯 → 如果是，以上总和 x2
7. 减去假烟的负分
8. 得出本轮得分
```

**要点：** 整个过程不需要任何辅助工具，心算即可。最大的一次乘法是大满贯的 x2，且只在极少数轮次发生。

### 8.5 为什么"杂货铺"很重要

杂货铺（6 张全不同品牌且全不同产地 → +8）是一个**兜底机制**。

在轮抽游戏中，经常出现"别人把你想凑的牌截走了"的情况。如果没有杂货铺，被截的玩家只能拿一堆散牌，得分极低，体验很差——特别是对新手来说。

杂货铺给了"什么都没凑到"的人一个回报：只要你的牌够散（6 张不同品牌+不同产地），+8 分的回报接近一个三条的 2/3。这保证了每个玩家每轮都有可行策略，降低了"废轮"的挫败感。

## 9. 假烟系统（社交核心）

### 9.1 设计理念

假烟是这个游戏**最独特的社交机制**。它不是传统卡牌游戏中的"惩罚牌"——它是一个心理战工具，一个话题制造机。

### 9.2 基础机制

- 牌堆中混入 **5 张假烟卡**
- 假烟卡的**卡背与普通烟盒卡完全相同**（红色），无法从背面区分
- 假烟卡的**正面看起来像高分卡**（印着知名品牌、看起来是 ★★★★ 或 ★★★★★），但右下角有一个小的「假」字标记
- 假烟卡的基础分 = **-5 分**

### 9.3 五张假烟

| 假烟卡 | 仿冒品牌 | 产地 | 伪装稀有度 | 风味文字 |
|--------|---------|------|-----------|---------|
| 假中华（硬） | 中华 | 上海 | ★★★★★ | 包装精美但烟丝发黑，拉丝不均匀 |
| 假熊猫（经典） | 熊猫 | 上海 | ★★★★★ | 熊猫印歪了，眼神呆滞 |
| 假芙蓉王（硬） | 芙蓉王 | 湖南 | ★★★★ | 金边掉色，封口粗糙 |
| 假黄鹤楼（1916） | 黄鹤楼 | 湖北 | ★★★★★ | 防伪码一刮就没，钢印模糊 |
| 假南京（九五） | 南京 | 江苏 | ★★★★ | 字体间距不对，老烟民一眼就认出来 |

### 9.4 为什么假烟看起来像高分卡

如果假烟一眼就能看出来（比如画风完全不同），那它只是一张"坏牌"——你看到就弃掉，没有任何社交价值。

但如果假烟看起来像"软中华"或"和天下"，社交就产生了：
- 你拿到手里第一反应是"好牌！"，然后仔细一看发现是假的——这个落差本身就好笑
- 你故意传给下家时，下家可能会犯同样的错误——"诶？这中华……等等，假的！"
- 计分时翻开假烟，全桌追溯"谁传的"——一个自发的社交推理环节

### 9.5 假烟在轮抽中的社交动态

**场景 1：你看到假烟，决定是否拿走**
- 自己吃下：你亏 5 分，但保护了后面的玩家（或者你单纯不想让下家因为没看清而中招）
- 传下去：下家可能中招，也可能看出来继续传——假烟在传的过程中制造持续的紧张感

**场景 2：多张假烟在同一轮**
- 如果 3 张假烟都出现在同一轮，牌堆变成了"雷区"，所有人选牌时都提心吊胆
- 这种随机性保证了每轮游戏的情绪基调不同

**场景 3：计分时翻开假烟**
- "是谁给我传的假烟？"——这个问题会让全桌进入"回忆 + 辩论"模式
- 坐你上家的人："不是我，我根本没看到" → 其他人："那肯定是再上家传过来的" → 混乱但快乐

### 9.6 "验货"机制（可选进阶规则）

**何时使用：** 当玩家已经熟悉基础规则、想要更多社交博弈时加入。第一次玩不建议使用。

**规则：**
- 每轮每人有 **1 次"验货"机会**
- 在选牌阶段的任意时刻，你可以指着**任意一位玩家最近一次扣下的牌**说"验货"
- 被验货的玩家必须**翻开那张牌**给所有人看
- 如果是假烟 → 假烟直接弃掉（不算任何人的分），验货成功
- 如果是真烟 → 验货失败，验货的人本轮 **-3 分**（冤枉好人的代价）

**验货产生的社交场景：**

- **读表情**："你刚才选牌的时候笑了一下，我要验货" → 全桌紧张
- **虚张声势**：故意表情做作让别人来验货，浪费对方的验货机会
- **时机博弈**：早验（信息少但可能抓到）vs 晚验（看了更多行为线索）
- **信任危机**："你验我？我们不是说好一起凑湖南套的吗？"

### 9.7 假烟的数值平衡

-5 分的惩罚是经过权衡的：

- 太轻（-2 分）→ 无所谓，不产生紧张感
- 太重（-15 分）→ 单张假烟就能毁掉一轮，挫败感太强
- -5 分 ≈ 一个对子 combo 的收益，意味着"中了一张假烟约等于少凑了一个对子"，痛但不致命

## 10. 道具牌系统

### 10.1 设计理念

道具牌是原版"烟灰缸"系统的极简化版本。保留了"被动加成"和"策略选择"的乐趣，但砍掉了所有复杂的触发条件和状态追踪。

### 10.2 核心规则

- 道具牌混在轮抽牌堆里，但使用**蓝色卡背**，发牌时就能看到这轮有没有道具
- 你选了道具牌 = 少选了一张烟盒 → 这是真实的成本
- 道具牌效果在卡面上用一句话写清，不需要查规则
- 每人每轮最多保留 **1 张道具牌**（防止囤积道具不拿烟盒的极端策略）

### 10.3 道具牌列表

| 道具牌 | 效果 | 社交属性 | 设计意图 |
|--------|------|---------|----------|
| **Zippo 打火机** | 你本轮分数最高的单个 combo +5 | 低（自己算） | 稳定收益，适合保守型玩家 |
| **茅台酒** | 指定 1 位其他玩家，你和 TA 各 +4 | **高**（选谁合作？） | 强制社交——"我请你喝酒" → 产生临时同盟 |
| **麻将桌** | 你的"老乡见老乡"从 +8 变为 +13 | 中 | 增强产地 combo 策略 |
| **澳门赌场** | 抛硬币：正面 +10，反面 -5 | **高**（全桌围观） | 纯气氛牌，紧张 + 刺激 |
| **假烟鉴定书** | 免疫 1 张假烟的 -5 分 | 中 | 反制牌，给被假烟困扰的玩家一个出路 |
| **黑市通行证** | 从本轮弃牌堆（最后抉择时弃掉的牌）中拿回 1 张 | 中高（其他人会议论你拿了什么） | 制造"垃圾堆里翻到宝"的惊喜 |
| **走私渠道** | 计分前，你可以将 1 张手牌和另一位玩家的 1 张手牌交换（双方公开） | **高**（谈判、威胁、交易） | 最社交的道具牌 |
| **控烟办罚单** | 指定 1 位玩家，TA 本轮得分最高的 combo 减半（向下取整） | **高**（攻击性） | 破坏领先者，防止一家独大 |

### 10.4 道具牌的策略层

道具牌的存在给轮抽增加了一个维度：**要不要拿？拿哪张？**

- 拿道具 = 少一张烟盒 = combo 更难凑 → 道具必须提供足够的回报才值得
- Zippo 的 +5 大约等于一张 ★★★ 烟盒的基础分 → 略微正收益
- 茅台酒的双方各 +4 = 总共 +8 → 超过一张牌的价值，但需要社交谈判
- 澳门赌场的期望值 = 0.5x10 + 0.5x(-5) = +2.5 → 期望值低但方差大，纯赌狗牌

### 10.5 为什么用蓝色卡背

道具牌用不同颜色的卡背（蓝色），意味着**发牌时所有人就知道这轮有几张道具**。

这是有意设计的透明信息：
- "这轮有 2 张道具！"→ 所有人开始思考要不要抢
- 如果道具是隐藏的（同色卡背），玩家可能误拿不想要的道具，降低策略感
- 蓝色卡背也在视觉上丰富了桌面，不全是红牌

## 11. 数值平衡

### 11.1 设计哲学

**combo 分 > 基础分**——策略比运气重要。

一个好的数值体系应该让"聪明地选择便宜牌凑 combo"的玩家胜过"无脑拿最高基础分"的玩家。

### 11.2 基础分与稀有度

| 稀有度 | 价格区间 | 基础分 | 牌堆张数 | 定位 |
|--------|---------|--------|---------|------|
| ★★ | <=15元 | **2** | 30 (~22%) | 弱但价廉量多，combo 的基石 |
| ★★★ | 16-40元 | **3** | 72 (~54%) | 中坚力量，什么策略都能配 |
| ★★★★ | 41-80元 | **5** | 21 (~16%) | 高分但数量少，对子（5+5+5=15）回报不错 |
| ★★★★★ | 81元以上 | **7** | 11 (~8%) | 极高基础分，但牌堆里同品牌通常只有 1 张，几乎不可能凑品牌对子 |

### 11.3 策略回报对比（以 7 张保留牌为例）

| 策略 | 典型构成 | 估算分数 | 难度 | 适合玩家 |
|------|---------|---------|------|---------|
| **散牌高分** | 7 张 ★★★ 无 combo | 7x3 = **21** | 低 | 新手 |
| **单对子** | 1 对品牌 + 5 散牌 | 2+2+5 + 5x3 = **24** | 低 | 新手 |
| **双对子** | 2 对品牌 + 3 散牌 | (2+2+5)x2 + 3x3 = **27** | 中 | 中级 |
| **三条** | 1 三条 + 4 散牌 | 2+2+2+12 + 4x3 = **30** | 中 | 中级 |
| **产地套** | 3 同产地 + 品牌对 + 散牌 | 3+3+3+8 + (含2同品牌+5) + 2x3 = **31** | 中高 | 中级 |
| **三条+产地叠加** | 3 同品牌+同产地 + 4 散牌 | 2+2+2+12+8 + 4x3 = **38** | 高 | 高级 |
| **大满贯** | 上述 + 三种规格齐全 | ~42 x 2 = **~84** | 极高 | 赌狗 |
| **完全失败（吃假烟）** | 无 combo + 假烟 | ~18 - 5 = **13** | — | 倒霉蛋 |

**分数梯度：** 13 → 21 → 24 → 27 → 30 → 38 → 84

### 11.4 模拟验证结果

基于 10,000 局模拟测试的数据：

**平均每轮得分：**

| 人数 | 平均分 |
|------|-------|
| 3人 | 31 |
| 5人 | 40 |
| 7人 | 31 |

**策略平衡（5人局）：**

| 获胜策略 | 占比 |
|---------|------|
| BrandCombo（品牌 combo 为主） | 39.8% |
| RegionCombo（产地 combo 为主） | 29.7% |
| Balanced（均衡策略） | 20.8% |
| HighScore（高分散牌） | 9.7% |

**策略平衡（7人局）：**

| 获胜策略 | 占比 |
|---------|------|
| BrandCombo | 28.0% |
| RegionCombo | 25.8% |
| HighScore | 25.1% |
| Balanced | 20.2% |

**分析：** 品牌 combo 策略整体略优，但产地 combo 和均衡策略都有竞争力。7 人局中四种策略接近均衡，说明大桌游戏中没有绝对的最优策略。纯高分散牌策略在 5 人局中最弱，但在 7 人局中因牌堆更分散而回升。

### 11.5 假烟对平衡的影响

5 张假烟在 134 张烟盒牌中占约 3.4%（含假烟的总牌堆 147 张中占约 3.4%）。在 6 人 8 张/轮的游戏中，每轮约 48 张牌进入轮抽，期望看到 ~1.6 张假烟。

这意味着：
- 大多数轮次有 1-2 张假烟在流转——足够产生紧张感
- 不会出现"半数牌都是假烟"的极端情况
- 某些轮次可能 0 张假烟（所有假烟都在未使用的牌堆里）——这种不确定性本身就是乐趣

### 11.6 道具牌对平衡的影响

8 张道具在 147 张总牌中占约 5.4%。每轮可能出现 0-3 张道具。

最强道具（控烟办罚单、走私渠道）的效果是**相对的**——它们不是给你加分，而是改变玩家间的分差。这种零和效果在多人社交游戏中是健康的——它防止领先者一骑绝尘，保持游戏的悬念。

## 12. 卡面设计

### 12.1 设计原则

每张卡需要同时服务两个目的：
1. **功能性**：玩家扇开一手牌时，一眼就能看到关键信息（品牌、产地、规格、分数）
2. **情感性**：翻开每张卡时有小惊喜（风味文字、烟盒图的文化联想）

功能性优先——如果一个玩家需要把牌拿到眼前仔细看才知道这张牌是什么品牌、多少分，游戏节奏就被打断了。

### 12.2 烟盒卡布局

```
┌─────────────────────────────┐
│ ★★★★★              上海     │  ← 左上：稀有度星星（用颜色区分）
│                             │     右上：产地（大字 + 省份色块）
│      ┌─────────────────┐    │
│      │                 │    │
│      │    烟盒实拍图     │    │
│      │  (ciggies.app)   │    │
│      │                 │    │
│      └─────────────────┘    │
│                             │
│  中  华                     │  ← 品牌名（卡面最大的文字）
│  硬盒 · 经典款              │  ← 具体款式名
│ ───────────────────────── │
│  常规           ★★★★★       │  ← 左下：规格  右下：稀有度图标
│                             │
│  "国宴标配。"          7    │  ← 风味文字 + 基础分（大号加粗）
└─────────────────────────────┘
```

### 12.3 关键设计细节

**基础分：** 用大号加粗数字印在右下角。扇开手牌时（只露出每张牌的右下角），一眼就能看到所有牌的分数。

**产地色块：** 每个产地有固定颜色的小色块（云南=红、上海=金、湖南=橙、湖北=黄、江苏=紫、浙江=绿等）。即使不看字，颜色相同 = 同产地。这在扇牌时非常高效——"我有 3 个红色块，云南三条稳了"。

**品牌名：** 卡面上最大的文字。因为品牌对子/三条是核心 combo，品牌名需要在任何距离都能辨识。

**规格标识：** 左下角标注"常规"、"细支"或"中支"，配合不同的图标（常规=粗线条、细支=细线条、中支=中等线条），扇牌时即可识别规格。

**风味文字：** 放在最底部，不影响功能信息的快速识别。但玩家翻到新牌时会自然看到——"国宴标配"、"工人阶级的选择"——这些文字是情绪价值的载体。

**图片数据：** 每张卡牌的烟盒图和产品链接来自 ciggies.app（cards.json 中的 image_url 和 sku_url 字段），确保卡面图片的真实性和权威性。

### 12.4 假烟卡布局

假烟卡的布局和普通烟盒卡**几乎完全相同**，但有以下区别：

- 右下角基础分位置印 **「-5」**（红色）
- 卡面某处有一个小小的**「假」字水印**
- 品牌名下方印 **"仿冒品"** 小字

**设计意图：** 假烟在手牌中应该需要"第二眼"才能识别——第一眼看到"中华 ★★★★★"会高兴，第二眼看到"假"字才会反应过来。这个发现过程本身就是乐趣。

### 12.5 道具牌布局

```
┌─────────────────────────────┐
│            ◆ 道具 ◆         │  ← 明确标识为道具牌
│                             │
│      ┌─────────────────┐    │
│      │                 │    │
│      │     道具插画      │    │
│      │   （茅台酒瓶）    │    │
│      │                 │    │
│      └─────────────────┘    │
│                             │
│  茅  台  酒                 │  ← 道具名
│ ───────────────────────── │
│                             │
│  选 1 位玩家               │  ← 效果描述（一句话说清）
│  你和 TA 各 +4 分          │
│                             │
│  "酒逢知己千杯少。"         │  ← 风味文字
└─────────────────────────────┘
```

### 12.6 卡背设计

| 卡牌类型 | 卡背颜色 | 卡背图案 |
|---------|---------|---------|
| 烟盒卡 + 假烟卡 | **红色** | 中国传统云纹 + Cigga Go logo |
| 道具牌 | **蓝色** | 同款云纹但蓝色调 + "道具"字样 |

烟盒和假烟共用卡背是有意设计——如果假烟有不同的卡背，它就完全失去了社交价值。

## 13. 收藏体系与工艺

### 13.1 设计理念

关于收藏性与游戏性并重的整体理念、稀有度的工艺哲学、以及卡牌作为文化载体的定位，详见 [第 4 节：收藏设计理念](#4-收藏设计理念)。本节聚焦于具体的 SKU 级工艺规格。

### 13.2 工艺分层

#### 基础层：★★ 和 ★★★ 卡（102 张）

- **材质：** 标准 310gsm 铜版纸 + 哑光覆膜
- **工艺：** 标准四色印刷
- **手感：** 顺滑、轻薄，适合快速扇牌
- **设计意图：** 成本可控，这些是游戏中使用量最大的牌，需要耐用

#### 进阶层：★★★★ 卡（21 张）

- **材质：** 350gsm 特种纸 + 磨砂覆膜
- **工艺：** 烫银边框 + 局部 UV（烟盒图部分凸起光泽）
- **手感：** 明显比基础卡厚实，磨砂触感
- **具体 SKU 建议：**

| 卡牌 | 特殊工艺建议 | 理由 |
|------|-------------|------|
| 南京九五 | 烫金"九五"二字 | 呼应"九五之尊"的霸气 |
| 南京·金陵十二钗 | 烟盒图部分做浮雕 | 十二钗人物画适合凹凸工艺 |
| 荷花 | 烟盒图部分做局部 UV | 荷花图案在光泽下更生动 |
| 黄金叶·天叶 | 烫金叶片图案 | 名字里就有"金叶" |
| 冬虫夏草 | 磨砂 + 触感漆（模拟中药质感） | 主题呼应 |

#### 传说层：★★★★★ 卡（11 张）

- **材质：** 400gsm 棉质特种纸 + 丝绒覆膜
- **工艺：** 烫金边框 + 全息烟盒图 + 限量编号（如 "003/500"）
- **手感：** 厚实、丝绒触感——摸起来就知道是"好牌"
- **具体 SKU 建议：**

| 卡牌 | 特殊工艺 | 理由 |
|------|---------|------|
| 中华·硬盒 | 烫金边框 + 经典红金配色全息 | 中国烟草的图腾，必须是收藏级 |
| 中华·听装 50 支 | 金属质感覆膜 | 呼应铁听包装 |
| 熊猫·经典 | **浮雕工艺**（熊猫图案凸起） | 国宝级别，指尖能摸到熊猫 |
| 大重九 | 烫金 + 镭射渐变边框 | 顶级云烟的代表 |
| 和天下 | 烫金 + 暗纹（隐藏的祥云图案，特定角度才能看到） | 呼应"天下"的大气 |
| 黄鹤楼 1916 | 复古烫铜 + 做旧边角 | 百年经典的历史感 |

### 13.3 扩展包的收藏价值

| 扩展包 | 内容 | 工艺特色 | 收藏定位 |
|--------|------|---------|---------|
| **国际品牌扩展包** | 21 张国际品牌 | 统一烫银，模拟机场免税店的高端感 | 补全收藏 + 解锁 8 人 |

## 15. 扩展性设计

### 15.1 产品路线图

```
基础版（147 张牌，支持 3-7 人）
  ├── 国际品牌扩展包（+21 张，解锁 8 人游戏）
  ├── 收藏限定包（+12 张历史/停产品牌，稀有度 +1）
  ├── 地域扩展系列
  │     ├── 云南卷（+12 张）
  │     ├── 湖北卷（+12 张）
  │     └── 上海卷（+10 张）
  └── 赛季限定（联名 / 节日主题）
```

### 15.2 国际品牌扩展包

21 张国际品牌（万宝路、骆驼、三五、大卫杜夫、柔和七星、爱喜、高希霸、登喜路等），来自 6 个国家/地区。加入后总牌数达到 168 张，支持 8 人 x 7 张/轮 x 3 轮 = 168 张。

**专属 combo：**

| Combo | 条件 | 加分 | 设计意图 |
|-------|------|------|----------|
| **免税店** | 3 张国际品牌 | +12 | 类似于产地 combo 的国际版 |
| **中西合璧** | 1 张中国 ★★★★★ + 1 张国际品牌 | +8 | 鼓励混搭 |

### 15.3 收藏限定包

12 张历史/停产品牌卡牌，享受 **+1 稀有度收藏溢价**。

包含品牌：飞马（极度稀缺——ciggies.app 上仅 1 个 SKU）、上海（品牌）、阿诗玛、小熊猫、石林、雪莲等。

**设计意图：** 这些品牌承载了中国烟草史上的文化记忆，很多已经停产或极难买到。+1 稀有度加成让原本 ★★ 的牌变为 ★★★（基础分 3），增加了游戏中的价值和收藏动力。

### 15.4 地域扩展系列

每个地域包增加该省份的卡牌密度，使产地 combo 策略在该局游戏中更强。

| 扩展包 | 张数 | 效果 |
|--------|------|------|
| 云南卷 | 12 | 云南牌密度大增，凑同乡/老乡见老乡更容易 |
| 湖北卷 | 12 | 湖北牌密度大增，黄鹤楼系列更丰富 |
| 上海卷 | 10 | 上海牌密度大增，中华/熊猫系列更丰富 |

**设计理念：** 加入地域包后该省份的产地 combo 变得更容易凑，但也意味着更多人会竞争同一产地——策略动态会因此改变。每局游戏可以选择加入 0-2 个地域包来调节体验。

### 15.5 设计预留

以下机制在基础版中**不实现**，但设计时预留扩展空间：

- **团队模式（2v2 / 3v3）**：可以通过"茅台酒"类道具的方向扩展
- **Boss 挑战模式**：合作对抗 Boss 卡的变体规则
- **收藏册**：实体收藏册，按品牌/产地/规格分页，集齐一页有奖励
- **锦标赛规则**：多局淘汰制，适合线下活动

### B. 从数字到实体：设计转化记录

本附录记录了 Cigga Go 从数字版（单人 roguelike）到实体卡牌版（多人聚会游戏）的关键设计转化过程。这些决策构成了实体版设计的基础，保留在此作为设计溯源和参考。

#### B.1 单人 → 多人

**决策：** 竞争型轮抽（所有人同时行动，比总分）

**为什么不是合作型？**
- 合作型（一起打 Boss）需要更复杂的规则来模拟 Boss 行为，增加了教学成本
- 烟草主题天然有竞争感——"谁的烟好"是中国烟民文化里的日常话题

**为什么不是回合制（一人一回合）？**
- 8 人回合制意味着你要等 7 个人操作才轮到自己，游戏时长会膨胀到 40+ 分钟
- 同时行动是支撑大人数 + 短时长的唯一方案

#### B.2 复杂乘法 → 纯加法（+ 唯一乘法大满贯）

**决策：** 所有 combo 给固定加分，只有"大满贯"（同品牌 3+ 张且覆盖三种规格）触发总分 x2

**为什么？**

数字版的公式是 `基础分 x 稀有度倍率 x 品牌Combo x 产地Combo`，多层嵌套乘法。这在电脑上自动计算没问题，但线下 8 个人同时算分就崩溃了：

- 加法：心算 1 秒 → "2+3+3+5=13"
- 乘法：心算 10 秒 → "13 乘以 1.5 再乘以 1.3 等于...你等我一下"
- 最慢的那个人算完之前，其他 7 个人都在干等

**为什么保留大满贯 x2？**

一个游戏需要"炸裂时刻"。如果所有东西都是加法，最高分和最低分差距不大，缺少高潮。大满贯的 x2 是整局游戏的"核弹"——极难达成，但一旦达成全桌沸腾。

而且只有一个乘法点，计算不会复杂：先正常加法算完总分，如果有大满贯就翻倍，完事。

#### B.3 四维 → 三维

**决策：** 砍掉"年代"维度，用"规格"（常规/细支/中支）替代

**为什么？**
- 年代信息对非烟民来说缺乏辨识度——"这包烟是 00s 还是 10s 的？"大部分人无法判断
- 规格（常规/细支/中支）是烟支的物理特征，直观可见，更容易理解
- 三种规格近乎均匀分布（45/45/44），为大满贯提供了更合理的设计空间
- 减少一个维度降低了选牌时的认知负担，同时保持了足够的 combo 深度

#### B.4 Roguelike 构筑 → 固定三轮制

**决策：** 砍掉商店、牌组构筑、run 结构，改为 3 轮独立轮抽

**为什么？**

Roguelike 的核心乐趣是"在一次 run 中逐渐变强"，但这需要：
- 持续的状态追踪（你的牌组、你的升级、当前关卡）
- 渐进的难度曲线
- 30-60 分钟的时间投入

这些都和"10-25 分钟聚会游戏"矛盾。三轮制保留了"越来越了解牌堆构成"的学习曲线（第一轮探索，第二轮开始有策略，第三轮全力冲刺），但不需要任何跨轮状态追踪。

#### B.5 烟灰缸系统 → 道具牌

**决策：** 不做独立的烟灰缸系统，改为 8 张道具牌混入轮抽牌堆

**为什么？**

数字版有 50 个烟灰缸，各有不同触发条件和加成。实体化的问题是：
- 50 个效果意味着玩家需要频繁查规则书
- 被动效果需要每次计分时记住并应用
- 独立的烟灰缸槽位增加了组件和规则

把核心效果浓缩成 8 张道具牌，混在轮抽里，解决了所有问题：
- 效果一句话写在卡面上，不用查规则
- 选了道具就少选一张烟盒，有真实的取舍
- 不需要额外组件或规则

详见 [第 10 节：道具牌系统](#10-道具牌系统)。
